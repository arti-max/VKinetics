# API: Утилиты (`utils.lua`)

Вспомогательные функции для работы с векторами, матрицами и установкой блоков.

## `smart_placement(x, y, z, playerid)`

Умная установка блоков. Анализирует вектор взгляда игрока (`player.get_dir`) и состояние приседания (`is_crouching`), чтобы корректно установить вращение блока (ось X, Y или Z). Записывает дополнительные данные о повороте в кастомное поле `roll`.

## Генерация осей

* `get_perp_offsets(axis)`: Возвращает перпендикулярные смещения для указанной оси. Полезно для шестеренок, передающих вращение боком (с инверсией направления).
* `get_diag_offsets(axis)`: Возвращает диагональные смещения. Используется большими шестернями (`LargeCogwheel`) для угловых соединений с малыми.

## ECS Хелперы

* `make_entity_prefix(name)`: Формирует строку вида `voxel_kinetics:name_entity`.
* `make_position_offs(x, y, z, offs)`: Сдвигает координаты блока (обычно на `0.5`, чтобы центрировать модель сущности в блоке).
