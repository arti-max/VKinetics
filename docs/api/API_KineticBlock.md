# API: `KineticBlock`

Базовый класс `KineticBlock` предоставляет основной интерфейс для работы с сетями, вращением и ECS-сущностями.

## Базовые свойства (Properties)

Переопределите эти значения в вашем дочернем классе:

* `is_generator` (bool): Является ли блок источником энергии (по умолчанию `false`).
* `base_capacity` (number): Производимая энергия (SU).
* `base_impact` (number): Потребляемая энергия. Если `rpm * impact > capacity`, сеть уходит в overstress.
* `base_rpm` (number): Базовая скорость вращения генератора.

## Жизненный цикл (Lifecycle)

* `on_placed()`: Вызывается при установке. Регистрирует блок в `KineticRegistry`. В дочерних классах здесь следует спавнить ECS-сущность и вызывать `apply_network()`.
* `on_broken()`: Вызывается при разрушении. Удаляет блок из регистра, разрушает сеть (`Pathfinder.destroy_network`) и удаляет привязанную сущность.
* `on_loaded()`: Вызывается движком при загрузке чанка. Восстанавливает стейт сети из сохраненных полей.
* `on_render(time, dt)`: Метод для обновления ECS-сущностей каждый кадр. Здесь происходит расчет текущего угла поворота (`get_visual_angle`) и обработка эффекта перегрузки сети (мигание костей).
* `on_tick()`: Вызывается каждый такт движка (20 TPS). Используется для таймеров (например, остановка `HandCrank`).

## Управление Сетью (Network Management)

* `apply_network()`: Ищет соседние кинетические блоки через `get_connections()`. Если находит существующую сеть, инициирует её пересчет (`Pathfinder.rebuild_network`). Если это генератор — создает новую сеть.
* `get_connections()`: Возвращает таблицу осей, к которым может подключиться блок.
* `can_connect_from(x, y, z)`: Проверяет, разрешено ли соединение с конкретных координат.

## Утилиты вращения и визуала

* `cache_rotation()`: Кэширует базовую матрицу поворота и оси (`_cached_offsets`, `_base_rot`) на основе текущего `block.get_rotation`.
* `get_visual_angle(gtime)`: Возвращает текущий угол поворота в градусах с учетом направления, RPM и глобального времени.
* `get_axis_name()`: Возвращает ось вращения (`"X"`, `"Y"` или `"Z"`).
