# Voxel Kinetics API

**Voxel Kinetics** - это фреймворк для движка VoxelCore, добавляющий кинетическую энергию, вращение и механизмы. В основе лежит графовая система сетей (Networks), которая динамически рассчитывает передачу вращения (RPM), нагрузку (Impact) и ёмкость (Capacity).

## Архитектура

Система разделена на три слоя для оптимизации и обхода ограничений статических вокселей:

1. **Блоки (Логика и Стейт):** Хранят состояние сети, направление, передаточное число (`ratio`) и UID привязанной сущности через поля блока (`fields`).
2. **ECS Сущности (Визуал):** Cтатические энтити со скелетом. Отвечают исключительно за плавное вращение и рендер костей.
3. **Граф сети (Pathfinder & Network):** Абстрактная математическая модель, которая пересчитывается (rebuild) при добавлении/удалении блоков, определяя перегрузку (overstress) и скорость вращения.
